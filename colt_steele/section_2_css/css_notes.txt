BASIC FORMAT
selector {
  property: value;
  another property: value;
}

Three types:
- element e.g. li {}
- class e.g. .special
- id e.g. #special

for group style application
class [multiple elements on page] vs id [unique element on page]

other important selectors:
- descendant selector e.g. h4 ul {} /ul child of h4/
- adjacent selector e.g. h4 + ul {} /ul next to h4/
- attribute selector e.g. input[style="text"]
- nth of type selector e.g. ul:nth-of-type(3) {} / ul:nth-of-type(even) {}
